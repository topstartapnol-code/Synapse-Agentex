
# 🌐 Веб-интерфейс для Telegram-бота "Тукан"

## Запуск

```bash
# Установите зависимости
pip install -r ../requirements-web.txt

# Запустите веб-сервер (он использует ту же БД, что и бот)
python server.py
```

Сервер запустится на `http://localhost:8080`.

## Использование

1. Сначала напишите боту в Telegram любое сообщение (чтобы создалась запись в БД)
2. Узнайте свой Telegram ID (через @userinfobot или @getmyid_bot)
3. Откройте `http://localhost:8080` в браузере
4. Введите ваш Telegram ID и войдите

## Функции веб-интерфейса

- 📁 **Папки** — создание, просмотр, переключение между папками
- 📝 **Заметки** — создание текстовых заметок
- ✅ **Задачи** — создание задач с отметкой выполнения
- 🔍 **Поиск** — быстрый поиск по всем заметкам
- ⏰ **Напоминания** — установка напоминаний через модальное окно
- 🎨 **Современный UI** — адаптивный дизайн, тёмная боковая панель
