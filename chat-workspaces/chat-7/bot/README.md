
# 🤖 Тукан-like Telegram Bot + Веб-интерфейс

Telegram-бот для управления задачами, заметками и напоминаниями, вдохновлённый ботом **Тукан** из статьи на Habr.

## Возможности

### Telegram Bot
- 📁 **Папки** — создавайте папки для сортировки заметок
- 📝 **Заметки** — сохраняйте текст, фото, видео, голосовые, документы
- ✅ **Задачи** — отмечайте задачи как выполненные
- 👥 **Группы** — добавляйте бота в чаты для совместной работы
- ⏰ **Напоминания** — устанавливайте напоминания на задачи
- 🎯 **Простой парсинг** — просто напишите «напомни завтра в 10:00 купить молоко»

### Веб-интерфейс (на FastAPI + Jinja2)
- 📁 Боковая панель с папками
- 📝 Создание/просмотр/удаление заметок
- ✅ Управление задачами с отметкой выполнения
- 🔍 Поиск по заметкам
- ⏰ Установка напоминаний
- 🎨 Современный минималистичный дизайн

## Установка и запуск

### Бот
```bash
cd bot

# Установить зависимости
pip install -r requirements.txt

# Создать .env файл с токеном
echo "BOT_TOKEN=your_telegram_bot_token_here" > .env

# Запустить бота
python main.py
```

### Веб-интерфейс
```bash
# Установить зависимости для веба
pip install -r requirements-web.txt

# Запустить веб-сервер (использует ту же БД)
cd web
python server.py
# Открыть http://localhost:8080
```

## Как получить токен для бота
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте полученный токен в `.env`

## Как попасть в веб-интерфейс
1. Напишите боту любое сообщение (чтобы создалась запись в БД)
2. Узнайте свой Telegram ID (бот @userinfobot)
3. Откройте `http://localhost:8080` и введите ID

## Структура проекта

```
bot/
├── main.py              # Точка входа бота
├── config.py            # Конфигурация
├── database.py          # Работа с БД (SQLite)
├── keyboards.py         # Клавиатуры для Telegram
├── scheduler.py         # Планировщик напоминаний
├── handlers/            # Обработчики команд бота
├── web/                 # Веб-интерфейс
│   ├── server.py        # FastAPI сервер
│   ├── static/          # CSS и JS
│   └── templates/       # HTML шаблоны
├── requirements.txt     # Зависимости бота
└── requirements-web.txt # Зависимости веба
```
